Right ⌘, right ⌥ or right ⌃ — your pick. Esc cancels, accidental taps are ignored, and ⌘-shortcuts keep working.

Install

From clone to dictating in a minute

Requires macOS 14+ (Apple Silicon recommended) and the Xcode command line tools.

$ git clone https://github.com/daviddao/murmur.git
$ cd murmur
$ make install   # builds Murmur.app → /Applications
$ open /Applications/Murmur.app

First run: allow Microphone + Accessibility, let the Parakeet model download once (~470 MB, cached). Full guide in the README.
